Rise 'n Shine

Deze code zorgt ervoor dat een lamp supersnel en superfel gaat knipperen als je niet binnen de tien minuten op de andere plek bent

Penalty

In de code die je hier ziet worden de variabelen aangemaakt en wordt de penalty functie geschreven. Deze functie zorgt ervoor dat er een geluid wordt afgespeeld als je niet optijd je bed uit bent ook stuurt deze een infrarood signaal die de bokshandschoen naar beneden laat komen. Met de A-Button zet je de tijd van de wekker.

Timer

In deze code gebruik je de B-Button om de wekker te starten zodra deze timer 0 is gaat de wekker af en wordt er een timer gestart. Als je niet binnen de penalty-tijd op de knop op de andere locatie drukt, wordt de functie op true gezet en wordt daarmee het infrarood signaal gestuurd. Ook heb je hier code die de timer aftelt.
